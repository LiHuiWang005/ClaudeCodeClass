## 二、会话主线

```
环境就绪 → 第一案例(泰坦尼克 EDA) → 进阶可视化(特征工程)
   → 案例库梳理 → 选定第二案例(Telco Churn) → 下载数据
   → 产品化(交互式网页看板) → 会话复盘归档
```

一条从"单个数据集分析"逐步走向"教学案例库 + 可交付网页产品"的递进式工作流。
核心能力线：**数据清洗 → 分组对比 → 特征工程 → 多维可视化 → 商业洞察 → 网页交付**。
