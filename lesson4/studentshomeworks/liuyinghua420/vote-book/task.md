## 执行建议

1. **Phase 1-6 已完成**：项目 MVP + 用户系统 + 管理员模式 + 密码找回 + 豆瓣封面自动抓取已全部实现。
2. **Phase 8**：7 个后续扩展方向，实际启动前需细化子任务和重新评估耗时。
3. **启动方式**：`node server.js` → 访问 `http://localhost:3000`。
4. **数据重置**：删除 `data/` 目录下所有 JSON 文件和 `.covers-fetched` 标记并重启服务即可。
5. **管理员机制**：系统预置管理员 `admin` / `123456`；首个注册用户也为 admin（若无预置）；后续用户为普通用户。
6. **邮件配置**：复制 `config.json` 并填入 QQ 邮箱和 16 位授权码；未配置时重置令牌直接展示在页面上。
