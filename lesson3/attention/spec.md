## 0. 设计原则（贯穿所有模块）

1. **每个互动都必须承载概念本身**，不是装饰。可视化超过 4-5 处就开始干扰阅读，违反"克制注意力"的主题。本网页严格控制在 7 个核心交互模块。
2. **三层结构强制**：每个模块都有「**互动 → 金句 → 思维模型**」三层。
   - 互动 = 让你**体验到**
   - 金句 = 让你**记得住**
   - 思维模型 = 让你**用得上**
3. **金句优先用原文**。原作者的话已经够锋利，再造反而稀释。需要替换时给"朋友圈传播版"做备选。
4. **思维模型固定格式**：每个模块底部固定区块，三行：
   - `【模型名】中文 / 英文`
   - 一句话定义
   - 何时调用 / 适用场景
5. **不要后端、不要构建工具、不要 npm**。纯 HTML + 原生 CSS + 原生 JS。所有依赖内联。
6. **移动端必须可读**。互动元素在窄屏自动降级为静态示意 + 解释文字。
7. **Apple 浅色克制美学**：大面积留白、巨型轻量级标题、单色照片级插画、零阴影或极淡阴影（`0 1px 2px rgba(0,0,0,0.04)`）、圆角统一 12-18px。"少即是多"——任何元素必须先通过"删掉它会怎样"的检验。

## 2. 七大核心模块详细设计

### 模块 1 — 注意力跷跷板

**论点**: 专注网络与默认模式网络是拮抗关系，分心是大脑的默认省电模式。

**互动设计**:
- 屏幕中央一个**水平跷跷板**，左端"专注"（Apple 蓝 `#0071e3` 光斑），右端"默认/分心"（Apple 橙 `#f56300` 光斑）。
- 用户**按住鼠标拖动跷跷板向左**（往专注侧倾斜）。
- 拖动时显示一个**单一数字计量**：`+0.30 卡路里 / 秒`（葡萄糖消耗，神经科学引用——专注时前额叶能耗）。数字实时累加，等宽字体 `tabular-nums`，避免跳动。
- 拖动持续 5 秒后，数字增速**自动减半**（每秒 +0.05 卡路里），下方一行小字浮出：`专注是会累的。`——加一层"长时间专注的真实感"体感。
- **松手瞬间**，跷跷板自动回弹到右侧（默认模式），配合一行冷感陈述：`松手 0.4 秒后，自动回到默认模式。`（不说教，让数字承载感受）
- 桌面端拖动 / 移动端长按。
- 静态降级（无 JS 或窄屏）: 直接展示左右对比插图 + 文案。

**金句**:
> **"专注是逆水行舟，分心是顺流而下。"**

**思维模型**:
- 【双网络拮抗模型】Task-Positive Network ↔ Default Mode Network
- 大脑两套网络此消彼长：专注网络主动耗能、默认网络省电待机。
- 何时调用: 当你"忍不住打开手机"时，提醒自己——这不是意志薄弱，这是默认网络在生效。

### 模块 2 — 多巴胺劫持循环

**论点**: 注意力被劫持的不是奖励本身，而是"不确定的预期"。

**互动设计**:
- 一个圆形按钮"刷一下"，外圈是循环动画：`触发 → 渴望 → 反馈 → 渴望加倍`。无音效（违反主题）。
- 用户每次点击：
  - **70% 概率** 出现普通卡片，从 5 类内容轮播：
    - 🐈 一张猫照片（SVG 几何插画，非真实图）
    - 📜 一句鸡汤（如"努力就有回报"）
    - 📰 一条假新闻（如"研究显示喝水能延寿"）
    - 😂 一个表情包文字版（如"打工人，加油"）
    - 🌅 一张风景图（SVG 简笔山/海/夕阳）
  - **30% 概率** 出现"超级奖励"卡片：金色闪光 + 一句**反讽文案**（这是整页爆点）：
    > "🎉 恭喜你！获得 +1 颗多巴胺。
    > 这条文案就是为了让你再点一次而写的。"
- 后台计数器记录**点击间隔时间**。当**连续 5 次点击间隔 < 1.5 秒**时（即真正进入劫持节奏），下方浮出一行小字：
  > "你已经在这个循环里点了 N 次，每次间隔 < 1.5 秒。这就是抖音、小红书、邮箱、微信对你做的事。"
- "我看明白了，停。"按钮**始终在场**：
  - 默认状态：弱视灰色 `#86868b`，字号小一号
  - 第 5 次点击后：变成 Apple 蓝 `#0071e3` 高亮（"等你回过神"的瞬间视觉反馈）
- **关闭后不消失**——按钮位置变为一行小字：
  > "你停下来了。这是这一整天里你为自己做的最重要的决定。"
  这是模块结束前的 punchline。

**金句**:
> **"你追逐的是预期，不是奖励本身。"**
> 备选 / 朋友圈版: "多巴胺贩卖的是'下一个会更好'的幻觉。"

**思维模型**:
- 【奖励预测误差陷阱】Reward Prediction Error Trap
- 公式: `触发 → 渴望 → 不确定反馈 → 更强渴望`。**不确定性**是劫持的终极武器。
- 何时调用: 任何"忍不住再刷一下"的产品（短视频/直播/游戏抽卡/邮箱通知/股票 App），其商业模式都嵌入了这个公式。

### 模块 3 — 管窥效应（认知带宽收割）

**论点**: 贫穷与忙碌通过"占用工作记忆"来掠夺注意力，强迫人做短视决策。

**互动设计**:
- 屏幕中央一张**纯文字平铺的生活全景**——大字号（28-32px）等距排布 6 个词：
  ```
  远方的山    家人    书架
  健身房     夕阳    长期目标
  ```
  Apple 风「克制美学」：不用图，纯文字 + 大留白，每个词都是 `#1d1d1f` 主文字色。
- 下方一排"焦虑事件按钮"——**砍到 4 个**最普世的：`房贷 / 工作邮件 / 父母身体 / 截止日期`。
- 用户**每点亮一个按钮**，全景区域受三层视觉叠加：
  1. **模糊**：CSS `filter: blur(2px → 4px → 6px → 8px)`（每点一个 +2px）
  2. **灰度**：`grayscale(0% → 33% → 66% → 100%)`
  3. **缩小裁切**：可视区从 100% 缩到 25%，外圈用 `mask-image: radial-gradient(...)` 渐变到 `#fbfbfd` 次级背景
  6 个词逐个变灰、逐个被裁出视野。
- 点到第 4 个（全部点亮）时，画面只剩中间一个小光圈，里面只剩「房贷 工作邮件 父母身体 截止日期」四个**焦虑词**清晰可见。下方一行：
  > "这就是你现在能看到的世界。长远规划、个人成长、真正重要的人，都在管子之外。"
- 一个"清空管窥"按钮，瞬间还原全景（300ms Apple 缓动），对比强烈。
- 静态降级：直接展示「全景态」和「管窥态」两张并置图 + 文案。

**金句**:
> **"贫穷不始于口袋的空虚，而始于注意力的溃散。"**

**思维模型**:
- 【认知带宽税】Cognitive Bandwidth Tax (Mullainathan & Shafir, 《Scarcity》)
- 缺钱 / 缺时间不直接致命，但会引发"管窥心态 (Tunneling)"：管内极清晰、管外漆黑，工作记忆被紧急事务占满，无余裕做长期决策。
- 何时调用: 当自己或他人在做"明显短视的决定"时，先检查 Ta 的认知带宽是否被占满，而非苛责"为什么不长远思考"。

### 模块 4 — 心流通道（两个旋钮）

**论点**: 心流不是被动等来的灵感，而是可以被主动设计的环境——只需调两个旋钮。

**互动设计**（**v0.3 重设计**：从 2D 图拖光点 → 双滑块，与思维模型「两个旋钮」原文呼应）:
- 两根水平滑块，左右刻度都是「低 ··· 高」：
  ```
  挑战难度  [────●────]    低 ─────────── 高
  当前技能  [──●──────]    低 ─────────── 高
  ```
- 滑块下方一段实时反馈区，根据两根滑块的相对位置切换文案 + Apple 语义色：
  - 挑战 >> 技能（差值 > 30%）→ Apple 红 `#ff3b30` ：
    > "你在**焦虑区**。任务超出能力，皮质醇飙升，注意力溃散。"
  - 技能 >> 挑战（差值 > 30%）→ Apple 紫 `#bf5af2` ：
    > "你在**无聊区**。技能溢出，多巴胺寻求外部刺激——开始想刷手机。"
  - 二者匹配 + 都在低位 → 中灰 `#86868b` ：
    > "你在**冷漠区**。两个旋钮都需要往上拨。"
  - 二者匹配 + 都在高位（且挑战略 > 技能 5-15%）→ Apple 绿 `#30d158` + 字号 +20% ：
    > **"你在心流。这就是你寻找的状态。"**
    （金句直接在互动里浮出，整页最强体感时刻）
- 滑块下方加一句小字提示，让读者自我代入：
  > "你今天的工作处在哪个区？"
- 移动端降级：保留双滑块（原生 `<input type="range">` 就支持）。无 JS 时降级为 4 个区的静态网格 + 文案。
- 下方"如何主动进入心流通道"折叠面板保留，含三条原则: 跳一跳够得着的目标 / 消除分心 / 启动仪式。

**金句**:
> **"心流不是等来的灵感，是设计出来的环境。"**
> 备选: "心流是精神熵的反面 —— 让混乱的意识变得高度有序。"

**思维模型**:
- 【心流通道】Csikszentmihalyi Flow Channel
- 心流发生在挑战与技能**都处于高位**且挑战略微超出舒适区的狭长通道。
- 何时调用: 当你做事感到"焦虑"或"无聊"时，先调整**两个旋钮**——挑战难度、当前技能——而不是责怪自己"不够自律"。

### 模块 5 — 注意力损益表（四投向 vs 三陷阱）

**论点**: 每一次注意力选择都是一次微小的自我定义。投向决定你是谁。

**互动设计**（**v0.3 简化**：100 颗豆拖拽 → 单滑块百分比分配。论点不变，操作量从 100 次降到 1 次）:
- 屏幕顶部一行: **"你今天有 100 颗注意力豆。"**
- 中间一根单滑块，两端分别是复利投向与收割陷阱：
  ```
  [复利投向 ●──────── 收割陷阱]
      60%                  40%
  ```
  - **左端（Apple 绿）复利投向**: 自我成长 / 复利技能 / 真爱与意义 / 创造价值
  - **右端（Apple 红）收割陷阱**: 外部劫持（凑热闹/随大流/操别人的心） / 内部消耗（内在反刍/习惯性反驳） / 执行幻觉（伪生产/即时满足）
- 滑块实时计算（`tabular-nums` 数字）:
  ```
  今日复利产出: +60
  今日被收割值: -40
  净值: +20
  ```
- 滑块下方加一行小字锚定现实：
  > "这个比例和你真实的今天像吗？"
- 底部根据净值出对应反馈（v0.3 文案加狠）:
  - 净值 > 0（绿多）: **"你今天在塑造一个值得成为的自己。"**
  - 净值 < 0（红多）: **"你今天为别人的算法做了一天免费工。"**
  - 净值 = 0 ±5: **"你留在了原地。明天还有一次机会。"**
- 移动端降级：原生 `<input type="range">` 就能跨端工作。无 JS 时显示静态对比两栏文案。

**金句**:
> **"我的经验就是，我同意去注意的东西。"** —— 威廉·詹姆斯
> 备选: "你注意过的一切的总和，构成了你的自我。"

**思维模型**:
- 【注意力损益表】Attention P&L Statement
- 注意力是有限资源，每天的支出有正向（复利）和负向（被收割）之分。把每天结束当作一次"日结"。
- 何时调用: 一天结束前花 30 秒回想"今天的 100 颗豆撒在哪了？"——比任何 To-Do 工具都更接近本质。

### 模块 6 — 注意力即财富（复利 vs 被收割）★新增

**论点**: 注意力是比时间更根本的原始资本；不守护它，等于把生命无偿让人变现。

**互动设计**（**v0.3 重设计**：双轨小人动画 → 单柱状图能力曲线。收紧互动焦点，与模块 5 论点区分——M5 讲「一天的分配」，M6 讲「十年的复利差」）:
- 一个**双线柱状图 / 折线图**（纯 SVG inline，CSS keyframes 动画），X 轴 = 时间，Y 轴 = 能力 / 产出：
  ```
    能力 ▲
         │                                    ╱── 复利（绿）
         │                          ╱╱╱
         │                ╱╱╱
         │       ╱╱╱                      ─── 被收割（红）
         │ ══════════════════════════════
         └───────────────────────▶ 时间
           1天    1月    1年   5年   10年
  ```
- 两条曲线视觉对比：
  - **复利曲线（Apple 绿 `#30d158`）**：指数上升。第 1 天几乎平，第 1 月微翘，第 1 年明显上升，第 10 年陡峭。
  - **被收割曲线（Apple 红 `#ff3b30`）**：始终平或微降。10 年后与复利曲线的差距触目惊心。
- 用户拖动**时间滑块**（1 天 / 1 月 / 1 年 / 5 年 / 10 年五档），图表高亮对应时间点，下方数字同步更新：
  ```
  时间：10 年后
  复利路径产出：~ 3,650 次深度工作
  被收割路径产出：~ 3,650 次刷屏
  能力差距：约 100 倍
  ```
- 滑到 10 年时，下方浮出金句：
  > **"时间一样长，注意力的去向决定了你站在哪里。"**
- 互动区下方保留**对比表格**（论点收尾，不再和动画并置）:
  | 维度 | 复利路径 | 被收割路径 |
  |---|---|---|
  | 深度工作能力 | 越来越强 | 退化到无法忍受无聊 |
  | 产出独特性 | 不可替代 | 随时可被替换 |
  | 时间杠杆 | 一份投入持续产出 | 一份投入一次性消耗 |
  | 选择权 | 越来越多 | 越来越少 |
- 移动端降级：滑块保留（原生 range），图表用响应式 SVG `viewBox` 自动缩放。无 JS 时显示"10 年态"静态 SVG + 对比表格。

**金句**:
> **"贫穷不始于口袋的空虚，而始于注意力的溃散。"**
> 备选 1: "时间是载体，注意力才是装在里面的黄金。"
> 备选 2 (传播版): "你不守护注意力，就要付费让别人收割你。"

> 注: 此金句与模块 3 同源。模块 3 的"贫穷"指向**当下的认知贫困**（管窥），模块 6 的"贫穷"指向**长期的生产性贫困**（无法深度产出）。两处都用，是有意的回声结构 —— 第二次出现时读者会理解得更深。

**思维模型**:
- 【注意力资本论】Attention as Primary Capital
- 三层因果链：
  1. 注意力 = 比时间更根本的原始资本（时间是载体，注意力是黄金）
  2. 注意力变现 = 现代商业核心模式（你不投资自己，就被别人投资，赚你的钱）
  3. 财富 = 长期专注的复利产物（深度工作 → 独特产出 → 不可替代价值 → 自由）
- 何时调用:
  - 每次想下载一个"很爽但没用"的 App 之前
  - 每次评估一份工作 / 一段关系 / 一个项目时——问"它会让我注意力复利还是注意力损耗？"
  - 当你觉得"我已经很努力了为什么没结果"时——大概率是注意力撒在了浅层、易替代的事上

### 模块 7 — 注意力账户清零（向死而生）

**论点**: 死亡的必然性赋予每一个当下选择以重量。

**互动设计**:
- 一个**输入框**: "你的年龄是 ____ 岁"
- 计算并显示（假设寿命 80 岁，扣除睡眠按 1/3 计）:
  - **主显示（冷感克制）**: 用 `tabular-nums` 等宽数字呈现
    ```
    你还有约 18,250 天
    ```

### 4.1 色板（Apple 浅色 · Light Appearance）

参考 Apple Human Interface Guidelines (Light) 与 apple.com 产品页系统色。

| 用途 | 色值 | 含义 |
|---|---|---|

| 次级背景 | `#fbfbfd` | 区块/分节背景（苹果官网经典淡灰白） |
| 表面/卡片 | `#f5f5f7` | 卡片底色（System Gray 6 浅色） |
| 边框/分隔 | `rgba(0,0,0,0.08)` | 极淡边框，仅在必要时使用 |
| 主文字 | `#1d1d1f` | 接近纯黑（apple.com 标题色） |
| 次文字 | `#6e6e73` | 中灰（副标题、说明文字） |
| 弱文字 | `#86868b` | 浅灰（注释、metadata） |

| 警示红 | `#ff3b30` | Apple Red（陷阱 / 收割 / 焦虑区） |

**强调色使用克制**：所有彩色都仅出现在「数据/状态/语义」位置，不用于装饰。文字 99% 是 `#1d1d1f` 或 `#6e6e73`。

### 4.2 字体

- 全局字体栈：`-apple-system, BlinkMacSystemFont, "SF Pro Display", "SF Pro Text", "Helvetica Neue", "PingFang SC", "Noto Sans SC", sans-serif`
- 等宽（数字与代码）：`"SF Mono", ui-monospace, Menlo, Consolas, monospace`
- 字重梯度（Apple 风：大字偏轻，小字偏重）：
  - Hero 巨标题：`font-weight: 600`，字号 `clamp(56px, 9vw, 112px)`，`letter-spacing: -0.04em`
  - 章节标题（H2）：`font-weight: 600`，字号 48-64px，`letter-spacing: -0.02em`
  - 模块小标题（H3）：`font-weight: 600`，字号 28-32px
  - 正文：`font-weight: 400`，字号 17-19px，`line-height: 1.47`
  - 金句：`font-weight: 600`，字号 +30%，颜色 `#1d1d1f`，无引号装饰，居中大字呈现
  - 说明/弱文字：`font-weight: 400`，色 `#6e6e73`
- **数字使用 `tabular-nums`**（如倒计时、剩余小时数），避免跳动。

### 4.3 排版与节奏

- 内容最大宽度 `980px`（与 apple.com 产品文章页对齐），居中。
- 模块间垂直留白至少 `120px`（桌面）/ `80px`（移动）。
- 每个模块固定结构：`小标题 → 引文 → 互动区 → 金句 → 思维模型 → 长论证文字`。
- 长论证文字保持**克制**：每段不超过 4 行，避免与互动区视觉打架。
- **零边框为默认**：用空白和背景色对比（`#ffffff` ↔ `#fbfbfd` ↔ `#f5f5f7`）来分隔区块，而不是用线条。
- 圆角统一：卡片 `18px`、按钮 `980px`（pill 圆角，Apple 经典）、输入框 `12px`。
- 阴影极淡或无：仅在浮起卡片时使用 `0 4px 24px rgba(0,0,0,0.06)`。

### 4.4 动效原则

- 所有动效**默认就绪、不抢夺**。仅在用户主动操作时（拖、点、滚）激活。
- 缓动曲线统一使用 Apple 系：`cubic-bezier(0.4, 0, 0.2, 1)`，时长 `300-500ms`。
- 自动循环动画（如多巴胺循环）必须有"暂停 / 我看够了"按钮——本网页主题不允许劫持读者注意力。
- 滚动时章节标题以 Apple 风格的"渐入上移"出现（`translateY(20px) → 0` + `opacity 0 → 1`）。
- 尊重 `prefers-reduced-motion`，关闭所有非必要动画。
